<--!KELOLA DATA BARANG-->

web ini digunakan untuk mengelola data barang / mendata barang
web ini dibuat dengan Notepad++,XAMPP, dan bisa diakses dengan web browser
di web ini kita dapat menginputkan data dan dilengkapi dengan edit hapus agar memudahkan pengguna
jika terdapat kesalahan dalam menginputkan data.

Cara menjalankan
1.aktifkan xampp (apache,mysql)
2.buka web browser
3.ketikkan localhost/ajaro(MILA)/home.php
4.lalu akan masuk pada tabel barang jika ingin menambah data barang klik button "Tambah"
5.dan dialihkan pada halaman input data isi data-data untuk menambahkan 
6.jika sudah berhasil makan akan muncul tulisan "Input data berhasil"

web yang dibuat ini tidak menggunakan framework karena,pelajaran web di sekolah baru sampai php native.
